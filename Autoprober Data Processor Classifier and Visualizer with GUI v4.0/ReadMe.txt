* - If you wish to use the Software with GUI, please download the ".zip" file and see the
    tips below:
    1 - Install Anaconda (Python 3), (https://www.anaconda.com/products/individual);
    2 - Open Anaconda Prompt and execute "conda install -c anaconda plotly" in order to
	install Plotly;
    3 - If the user also wishes to use the Sensor Classifier, then he should follow the 
	steps (a) and (b); otherwise, skip to item 4.:
	a. If your computer has an Nvidia graphics processing unit (GPU), I recommend
	   installing Keras/Tensorflow with GPU support.  You can open Anaconda Prompt
	   and execute "conda install -c anaconda keras-gpu";
	b. If your computer does not have an Nvidia GPU, you should install 
	   Keras/Tensorflow central processing unit based. You can open Anaconda Prompt
	   and execute "conda install -c anaconda keras";
    4 - After the installation, you may open the software and use it. You have a ".py"
     	and ".ipynb" files. Choose whatever integrated development environment you which,
	but I recommend using the Jupyter Notebook and hence the ".ipynb" file.

**All the packages needed are open-source.
**The data provided in the "Data unseen" folder was generated artificially simulating
  real-world data.
**Sometimes installing Keras/Tensorflow might be a challenge to you, if you need help
  contact one of the two emails bellow:
